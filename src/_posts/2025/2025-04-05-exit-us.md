## Suggestions for a more secure digital life

- Use a data-sensitive search engine (DuckDuckGo, Qwant)
- Use a VPN, when possible (wireguard / wirehole)
- Non-personalized email addresses per service
- Use a secure cloud (Nextcloud or pCloud)
- Use a password manager (KeePassXC)
- Use a friendly browser (Firefox)
- Use a secure messenger (Signal)
- Use a secure DNS
- Use Multi-Factor Authentication (MFA)
- Use a secure password (KeePassXC)
- Use a secure operating system (Linux)
- Opt-out of tracking (e.g. Google Analytics)
- Use an ad blocker (pi-hole/adguard)
- Do not use unknown cables or chargers
- Put smart devices in a separate network
