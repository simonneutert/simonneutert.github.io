This blog is about my journey as a software developer.\
Sharing insights, tips, and experiences from the world of coding and technology.

See what I see (my public photo album 📸) on
[Flickr](https://www.flickr.com/photos/simonneutert/) and get in touch with me
@[LinkedIn](https://www.linkedin.com/in/simon-neutert/)!\
I check my inbox almost quarterly 😅 🤞

I share my code on [GitHub](https://github.com/simonneutert) and help maintain
[shell-curry](https://shell-curry.simon-neutert.de/), an opinionated list of
terminal tools.

My passion is Ruby, yet I enjoy exploring what other languages bring to the
table.\
I've been dabbling with Clojure, exploring the
[land of LISP and parentheses](https://www.youtube.com/watch?v=HM1Zb3xmvMc&t=64s)
whenever I find the time 🥰

I run this website/blog using my custom static site generator, written in
TypeScript, with [Deno](https://deno.land/) as the runtime/ecosystem.\
The source code is available on GitHub:
[@simonneutert/deno-quickblog](https://github.com/simonneutert/deno-quickblog).
